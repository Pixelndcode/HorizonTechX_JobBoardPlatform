Job Board Platform

Overview

This project is a backend Job Board Platform developed using Flask and SQLite. It allows employers to post jobs and candidates to apply for available opportunities.

Features

- Job Listings
- Candidate Profiles
- Resume Management
- Job Applications
- Application Status Tracking
- SQLite Database Integration
- Flask Backend APIs

Technologies Used

- Python
- Flask
- SQLite
